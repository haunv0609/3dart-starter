# Installation
1. Install via Composer:
```bash
composer require haunv/3dart-starter
```
2. Publish file:
```bash
php artisan install:haunv
```
3. Edit file web.php in folder routes
```php
use Illuminate\Support\Facades\Route;


...
require  __DIR__  .  '/admin.php';
```
